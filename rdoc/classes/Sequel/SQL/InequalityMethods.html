module Sequel::SQL::InequalityMethods

  1. lib/sequel/sql.rb

This module includes the inequality methods (>, <, >=, <=) that are defined on objects that can be used in a numeric or string context in SQL.

Sequel.lit('a') > :b # a > "b"
Sequel.lit('a') < :b # a > "b"
Sequel.lit('a') >= :b # a >= "b"
Sequel.lit('a') <= :b # a <= "b"
