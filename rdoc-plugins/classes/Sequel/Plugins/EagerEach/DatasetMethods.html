all (&block)

If eager loading, clone the dataset and set a flag to let each know not to call all, to avoid the infinite loop.

[show source] 
# File lib/sequel/plugins/eager_each.rb, line 48
def all(&block)
  if use_eager_all?
    clone(:all_called=>true).all(&block)
  else
    super
  end
end
